EXECUTION STEPS
---------------1)Open Terminal2)Navigate to the project source file directory3)Execute command to start server – ‘python3 IRC_server_sockets.py  8000’.
4)Open another terminal session
5)Execute command to start client – ‘python3 IRC_client_sockets.py localhost 8000’.IRC commands:-
——————
\listrooms
\listusers <room-name>
\join <room-list>
\leave <room-list>\msg <message>
\msgroom <room-count> <room list> <message>
\prvmsg <user-name> <message>
\quit
